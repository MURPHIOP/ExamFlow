# AI/ML Plan (Advisory-Only)

AI features in ExamFlow are meant to reduce manual workload while keeping humans in control.

## Planned AI/ML Modules
1. OCR-assisted legacy form digitization
2. Duplicate detection (deterministic + fuzzy checks)
3. Name/spelling correction suggestions
4. Centre allocation recommendations
5. Fraud and anomaly flagging
6. Result analytics insights
7. Admin natural language assistant (read-only guidance/search)

## Critical Safety Rule
AI outputs are advisory. AI must **never** auto-approve, auto-reject, auto-delete, auto-publish, or perform destructive actions without explicit authorized human action.

## Current Status
No AI/ML logic is implemented in this foundation step.
