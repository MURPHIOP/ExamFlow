# UI Design System (Foundation)

## Design Goals
- Premium SaaS dashboard aesthetics
- Role-friendly and office-friendly usability
- Strong readability in both light and dark modes

## Color System
- Primary: violet/indigo accents
- Secondary semantic accents: cyan/emerald/amber
- CSS variables drive light/dark token values

## Layout
- Left sidebar navigation
- Top navbar with actions and theme toggle
- Responsive content grid with card-based sections

## Components
- Metric cards
- Section cards
- Status badges
- Skeleton loaders
- Empty and error states
- Table-ready layout blocks and chart placeholders

## Forms and Tables
- Clean spacing and clear labels
- Status badges with semantic color mapping
- Future-ready for filters, pagination, and validation feedback

## Motion
- Subtle transitions for theme switches and card interactions
- Framer Motion ready for phased animations

## Accessibility
- Contrast-conscious color pairs
- Keyboard focus visibility
- Reduced visual clutter and clear hierarchy
